# Atividade de Lógica e Estruturas Condicionais - GrowDev

## Descrição

Este repositório contém uma atividades simples de lógica de programação e estruturas condicionais. 
O objetivo desta atividade é proporcionar aos alunos uma prática sólida em conceitos fundamentais de lógica de programação e tomada de decisão por meio de estruturas 
condicionais.
